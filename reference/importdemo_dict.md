# Example data dictionary

Simulated data dictionary for the most common REDCap field types and
validations to be used with the sample data `importdemo_data` for
testing of `redcap_import_select` and `redcap_import_recode`. Item names
and coding show mismatches on purpose so that the interactive functions
can be tested.

## Usage

``` r
importdemo_dict
```

## Format

A data frame with 38 variable specifications.
