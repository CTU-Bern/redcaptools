# Example import data

Simulated import data for the most common REDCap field types and
validations to be used with the data dictionary `importdemo_dict` for
testing of `redcap_import_select` and `redcap_import_recode`. Item names
and coding show mismatches on purpose so that the interactive functions
can be tested.

## Usage

``` r
importdemo_data
```

## Format

A data frame with 17 observations and 32 variables.
